# The Mafia Barbershop Website

Website resmi untuk The Mafia Barbershop - barbershop premium di Surabaya dengan konsep maskulin & elegan.

## Struktur File

```
the-mafia-barbershop/
├── index.html          # File HTML utama
├── style.css           # File CSS terpisah (semua styling)
├── script.js           # File JavaScript terpisah (interaktivitas)
├── images/             # Folder untuk semua gambar
│   ├── logo.jpg
│   ├── outlet_lidah2.jpg
│   ├── outlet_mer.jpg
│   ├── pompadour.jpg
│   ├── coloring.jpg
│   └── beardtrim.jpg
└── .vscode/            # Konfigurasi VS Code
```

## Fitur

- **Responsive Design**: Menggunakan Tailwind CSS untuk layout yang responsif
- **Interactive Elements**: JavaScript untuk toggle menu, form booking, dan animasi
- **Premium UI/UX**: Desain dengan tema gelap, animasi smooth, dan efek visual
- **Booking System**: Form booking online yang terintegrasi dengan WhatsApp
- **Gallery**: Showcase hasil potongan rambut dan interior barbershop
- **Multi-outlet**: Informasi untuk 2 lokasi di Surabaya

## Teknologi

- **HTML5**: Struktur markup
- **Tailwind CSS**: Framework CSS untuk styling
- **Vanilla JavaScript**: Interaktivitas tanpa framework
- **Google Fonts**: Typography (Playfair Display & Barlow)

## Cara Menjalankan

1. Clone atau download repository ini
2. Buka terminal di folder project
3. Jalankan server lokal:
   ```bash
   python -m http.server 8000
   ```
4. Buka browser dan akses `http://localhost:8000`

## Penggunaan

- **Navbar**: Navigasi antar section dengan smooth scroll
- **Hero Section**: Informasi utama dan call-to-action
- **Services**: Daftar layanan dengan toggle untuk melihat lebih banyak
- **Barbers**: Tim barber dengan informasi spesialisasi
- **Gallery**: Portfolio hasil kerja
- **Booking Form**: Reservasi online via WhatsApp
- **Footer**: Informasi kontak dan lokasi

## Pengembang

Website ini dibuat dengan fokus pada performa dan user experience yang optimal.
